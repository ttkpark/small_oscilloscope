# 오실로스코프 만들기
## 목표
- 아날로그 2채널 입력 가능한 오실로스코프
- 디지털 입력 모드 구현 및 신호 분석이 되는 오실로스코프
- 노브 컨트롤, 로터리 인코더 컨트롤
- DSO138 오실로스코프보다 더 좋은, Proteus의 오실로스코프 같은 오실로스코프
- Battery-Powered, 충전 가능
- 속도는 ESP32 구성의 최대치 (기존 칩 쓰거나, 문제가 크면 ESP32-C6 사용)
